Como funciona a conversão?

As bases numéricas são a forma como representamos quantidades:

  • Decimal (Base 10): É o sistema que usamos no dia a dia, com dígitos de 0 a 9.
  • Binário (Base 2): O idioma nativo dos computadores, usa apenas os dígitos 0 e 1 para representar estados de ligado/desligado.
  • Hexadecimal (Base 16): Muito usado em programação (como para cores CSS) para encurtar binários gigantes. Usa dígitos de 0 a 9 e as letras de A a F.
